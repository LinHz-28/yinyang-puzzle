# 食妖陰陽師 · 希望之印

> 執朱筆 · 連四象 · 喚字成印

一個東風陰陽玄怪風格的網頁解謎小遊戲。以「連連看」為核心機制，玩家以朱筆連結四象之眼（朱雀、青龍、白虎、玄武），讓散落於符陣中的英文字母依序亮起，最終揭示答案 **HOPE**。

## 線上試玩

https://LinHz-28.github.io/yinyang-puzzle/

## 玩法

1. 滑鼠或觸控點選任一錨點（朱雀／青龍／白虎／玄武），再點另一錨點即可連線。也可直接拖曳。
2. 朱線經過的字母會以朱光亮起。
3. 連對四道符線，使亮起的字母正好為 **H、O、P、E** 即破關。

### 正解四線

| 連線 | 浮現字母 |
| --- | --- |
| 青龍 → 朱雀（左上斜線） | **H** |
| 青龍 → 白虎（橫貫線） | **O** |
| 玄武 → 朱雀（垂直線） | **P** |
| 白虎 → 朱雀（右上斜線） | **E** |

依此順序繪線，下方咒語欄會逐字浮現 `H → HO → HOP → HOPE`。

## 自訂

所有設定集中在 `index.html` 的 `<script>` 區塊：

- `ANCHORS`：四象座標與類型
- `LETTERS`：字母位置（前 4 個為正解，其餘為干擾字）
- `TOL`：字母被判定「在線上」的容忍距離（預設 14px）

若要更換題目／答案，重排 `LETTERS` 內容並調整位置即可。

## 技術

- 單一 HTML 檔，無相依
- SVG 渲染（含 Gaussian Blur 光暈 filter）
- Pointer Events（同時支援滑鼠與觸控）
